A.D.A.M. — Algorithmic Discourse And Memeing

A program that makes its own memes

*Early Development

Development

ADAM is not an AI, as it does not automatically learn, but it is an algorithm created to mimic popular meme styles.

Index

The algorithm uses an index of current events, pop culture, and internet humor. The algorithm pulls topics from the index, then continues to perform a variety of calculations to create the meme's content.

Emotions

The algorithm has a variety of emotions it can depict in the memes it generates. As part of the process, it determines an emotion before finalizing the text and image.
